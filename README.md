## 🌟 Vision

**Veds AI** is designed to provide an unparalleled AI interaction experience. Combining a stunning glassmorphic UI with robust, responsive capabilities, Veds AI is built for users who expect a seamless, beautiful, and intelligent interface.

## ✨ Features

- **Gorgeous Glassmorphic UI**: Experience a sleek, modern, and transparent design that feels incredibly premium.
- **Responsive by Design**: Works flawlessly on mobile, tablet, and desktop environments.
- **Seamless State Management**: Smooth transitions, smart history tracking, and intuitive user authentication flows.
- **Lightning Fast Performance**: Powered by Vite and React for near-instant rendering.

## 🚀 Quick Start

Follow these steps to get a local development environment up and running quickly.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/VedrajSingh1926/VedsAI.git
   cd VedsAI
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   *Your app will be running at `http://localhost:5173`.*
